# Rustyvibes-Gui

A Rust App that makes mechanical keyboard sound effects on every key press now with volume control!

# Installation

```
cargo build
```

Then find the binary in `debug/target`

# Usage

```
rustyvibes <volume> <soundpack_path>
```
